## Install

> [!NOTE]
> Linux: \
> `[sudo] ./intel-oneapi-xxxxx-toolkit-xxxxx_offline.sh -a --silent --cli --eula accept`

> [!NOTE]
> Определение переменных окружения под ОС Linux в sh скрипте: \
> `. /opt/intel/oneapi/<toolkit-version>/oneapi-vars.sh`
