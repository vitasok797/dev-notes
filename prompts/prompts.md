```
Приведи полную реализацию кода
```

## C++

```
Далее приведен код на языке C++ (стандарт C++20).
Проверь данный код.
Ищи ошибки или другие дефекты.
Предложи улучшения по производительности, упрощению или читаемости кода.
Укажи на неиспользуемые/недостающие заголовочные файлы.

<code>

Далее приведены тесты для кода.
Проверь данные тесты.
Укажи, есть ли неохваченные тестами пути кода или граничные случаи.

<tests>
```

```
Требования к коду на языке C++ приведены в файле `cpp-requirements.md`
```
